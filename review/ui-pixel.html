DUNGEON SHIFTPIXEL UI REVIEW · REV 04
NEW DIRECTION / PIXEL DIORAMA

像素地牢行动终端

保留 01 方案的克制层级，换成真正的像素界面系统：硬边框、分段状态、像素阴影、抖色材质与 RPG 档案结构。3D 角色和灯光保持原样，界面像一台记录异步劫案的旧掌机。

不是把画面变糊

像素化只发生在信息层。3D 场景继续承担空间、角色和灯光；UI 负责把危险、预算、技能和真人挑战压缩成清晰的游戏语言。

  • 像素字体只用于短标题、数值和按钮
  • 中文正文保持 16 px 清晰无衬线体
  • 状态变化采用分段条、硬闪和短 glitch
  • 所有功能图标仍是统一 SVG，不用 Emoji

VISUAL SYSTEM

01现代 3D × 复古 HUD

不对场景做全局像素滤镜。利用硬边 UI 与低多边形角色的共通几何感建立一致性。

02地牢专属色彩

琥珀只给宝物和奖励，青色只给入口与操作，玫红只给警戒，避免泛霓虹。
